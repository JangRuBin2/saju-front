2. **결제 페이지 번역** — FAQ, 소셜프루프, 기능 목록 텍스트가 한국어로 하드코딩됨. 다국어 번역 키로 교체 필요
3. **운세 카테고리 별점** — 현재 클라이언트 해시 기반 랜덤. 실제 API 응답에 카테고리별 점수가 포함되면 교체 필요
4. **행운의 색/숫자** — 마찬가지로 API 응답에 포함되면 교체 필요
5. **대시보드 최근 활동** — DB에서 UsageLog 조회 연동 필요
6. **BottomNav에 대시보드 추가** — 현재 5개 탭(홈/사주/운세/궁합/MY) 유지 중. 필요 시 추가
